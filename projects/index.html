Fallout 76 Decrypter

A simple console program for decrypting the weekly nuke launch for bethesdas fallout 76 game. Input the weekly codeword and the code piece letter / number pairs and the program will output all possible launch codes. The last set of codes found will also be stored and recalled upon relaunch of the program.

Wordlist Sorter

Ths program takes a word list in a ".txt" file and will create a new filtered word list based on give perameters such as the minimum and maximum number of letters and whether to exclude words with multiples of the same letter. You can also have the outputted list converted to all uppercase or all lowercase letters.

This program came about as a need to filter down the word list I was using in my Fallout76 nuke code decrypter.
